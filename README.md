## Requirements
Make sure your system has the correct Node and Npm versions.

- Node 6.2.x
- Npm 3.9.x

## Usage
- Changing source file in `/src` and click save.
- open `http:localhost:8000`.

## Install
```
npm install
```
or if already have yarn
```
yarn
```

## Start
```
npm start
```

## Build
```
npm run build
```
